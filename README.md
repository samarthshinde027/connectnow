# ConnectNow — Redesigned

## What's New in This Version
- ✨ Full UI redesign — minimal dark editorial aesthetic
- 🔤 Instrument Serif (display) + DM Sans (body) typography
- 🎨 Refined dark palette with purple/violet accents
- 🪄 Subtle animations and micro-interactions
- 💬 Redesigned chat with better message layout
- 🎯 Cleaner controls bar with icon + label
- 📱 Improved mobile responsive layout
- 🔘 Emoji reaction picker (no more prompts)
- 🌙 Light/Dark theme toggle

## Run Locally
```
npm install
node server.js
```

Open: http://localhost:3000

## Deploy to Render
- Build Command: `npm install`
- Start Command: `node server.js`

## Features
- Group WebRTC video calls
- Real-time chat with file sharing notifications
- Shared meeting notes (synced live)
- Raise hand
- Emoji reactions
- QR invite code
- Export attendance as CSV
- Password-protected rooms
- Host controls (mute/remove)
- Mobile responsive
